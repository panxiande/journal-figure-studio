# Archetype Profile Schema

Each archetype should define:

- Required data contract: minimal columns, units, grouping, repeats, source-image metadata.
- Visual encoding: x, y, color, shape, line, transparency, scale bar, colorbar, interval.
- Statistical annotation: raw points, n, error bars, CI, p values, effect size, test details.
- Layout constraints: legend placement, axis labels, annotation limits, panel labels, scale/color bars.
- Journal interactions: differences for final size, file type, fonts, and image integrity expectations.
- QA checks: archetype-specific failure modes.
