# General Publication Profile

Use when the user asks for a paper figure, SCI figure, high-level journal style, or does not know the final target journal.

## Rule Layers

### required

- rule: Do not alter scientific evidence images in ways that change interpretation.
  source_type: inferred
  confidence: high
  last_verified: 2026-06-16
  source_url: references/source-links.md
  applies_to: all

### recommended

- Use editable vector text for charts and annotations.
- Use RGB or grayscale unless the target journal says otherwise.
- Keep captions, author names, and manuscript titles outside the figure.
- Use colorblind-safe palettes and redundant encodings.

### style_observed

- Clean white background, restrained annotation, clear panel hierarchy, minimal decoration.

### unknown_or_verify

- Final width, height, file type, file size, dpi, and font restrictions depend on the target journal.

## Defaults

- Canvas: double-column 180 mm wide, maximum 170-220 mm high depending on content.
- Typography: Arial or Helvetica, 6-8 pt at final size, panel labels 8-9 pt bold.
- Export: SVG and PDF for editable figures; PNG preview; TIFF/PNG if submission target requires raster.
- Integrity: record source data and processing notes.
