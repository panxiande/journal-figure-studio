# Cell / Cell Press Profile

Use for Cell-family or Elsevier life-science figures when the exact Cell Press title is not specified.

## Rule Layers

### required

- rule: Final hard requirements vary by Cell Press journal and must be verified for the target title.
  source_type: publisher
  confidence: high
  last_verified: 2026-06-16
  source_url: https://www.cell.com/
  applies_to: all

### recommended

- Submit each figure as its own logical file and name files by figure number.
- Keep text concise; define symbols and abbreviations in the caption when possible.
- Embed fonts in vector formats or preserve editable text where accepted.
- Avoid using generative AI to create or alter scientific evidence images.

### style_observed

- Strong graphical narrative, polished schematic panels, clear hierarchy between mechanism, evidence, and quantification.
- More tolerant of schematic-led visual storytelling than stricter minimal profiles, while still requiring data traceability.

### unknown_or_verify

- Exact dimensions, acceptable formats, file size limits, and graphical-abstract requirements depend on journal and article type.

## Defaults

- Canvas: double-column 170-180 mm unless a target guide is provided.
- Typography: Arial/Helvetica, 6-8 pt body, bold panel labels.
- Export: PDF/SVG plus high-resolution PNG/TIFF; verify target title.
- Integrity: explicit AI-image and source-data compliance note.
