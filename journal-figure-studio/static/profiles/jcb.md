# Journal of Cell Biology Profile

Use for microscopy-heavy, cell biology, image plate, gel/blot, and JCB submission-style figures.

## Rule Layers

### required

- rule: Figures require sufficient resolution; use at least 300 dpi, with higher dpi for mixed line art or pure line art.
  source_type: official
  confidence: high
  last_verified: 2026-06-16
  source_url: https://rupress.org/jcb/pages/fig-vid-guidelines
  applies_to: all
- rule: Microscopy figures should include scale bars rather than only magnification.
  source_type: official
  confidence: high
  last_verified: 2026-06-16
  source_url: https://rupress.org/jcb/pages/fig-vid-guidelines
  applies_to: all

### recommended

- Maximum figure size around 7 in x 9 in.
- Use 8 pt text and about 5 pt for small corner labels when needed.
- Use RGB color and avoid outlined text.
- Provide source data and acquisition/processing metadata for microscopy and blot panels.

### style_observed

- Image plates should be organized in regular grids with consistent channel labels, scale bars, and matched quantification.

### unknown_or_verify

- Exact file formats and system checks should be verified against current JCB submission pages.

## Defaults

- Canvas: up to 7 in x 9 in.
- Typography: Arial/Helvetica, 8 pt body.
- Export: TIFF/EPS/PDF/SVG depending on panel type; retain editable labels where possible.
- Integrity: strict microscopy, gel/blot, source-data, scale-bar, and processing notes.
