# ACS / Chemistry Profile

MVP profile. Use for chemistry, materials, nanoscience, and spectroscopy figures, then verify the target ACS journal.

## Rule Layers

### recommended

- Make spectra axes, units, sample identifiers, and reaction/structure labels unambiguous.
- Preserve vector text and line art.
- Use color carefully for print and accessibility.

### style_observed

- Dense but clean panels, strong material/sample naming, clear inset management.

### unknown_or_verify

- Exact dimensions, type size, format, and graphical TOC requirements depend on the specific ACS journal.
