# IEEE / Engineering Profile

MVP profile. Use for engineering, signal processing, computing, and two-column conference/journal figures, then verify the target venue.

## Rule Layers

### recommended

- Design for narrow two-column layouts and grayscale-safe print.
- Use clear line styles and markers, not color alone.
- Keep axis labels and legends readable at final column width.

### style_observed

- Compact plots, strong legends, reproducible benchmark tables/plots, minimal decoration.

### unknown_or_verify

- Exact width, font, file type, and dpi must be checked for the specific IEEE venue.
