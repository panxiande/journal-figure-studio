# ML Conference Profile

MVP profile. Use for NeurIPS, ICML, ICLR, CVPR, ACL, and related benchmark/ablation figures.

## Rule Layers

### recommended

- Report seeds, folds, confidence intervals or variance, baseline fairness, metric definitions, and compute/runtime context when relevant.
- Avoid ranking plots that hide uncertainty.
- Use direct labels and compact legends that survive paper-column width.

### style_observed

- Dense line charts, ablation matrices, leaderboards, qualitative examples paired with quantitative summaries.

### unknown_or_verify

- Venue template sizing and camera-ready constraints must be checked against the current call for papers or author kit.
